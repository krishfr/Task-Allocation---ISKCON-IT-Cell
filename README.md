Task Allocation — ISKCON IT Cell

Web application for managing and allocating tasks within the ISKCON IT Cell team. Designed to help organisers assign work, track progress, and centralise task workflows.

Overview

Task Allocation project provides a platform for team members to view, assign, and manage tasks for ISKCON IT Cell activities. It improves coordination and workflow transparency for volunteers and organisers.

Tech Stack

Frontend React
Backend Node.js, Express.js
Styles HTML and CSS

Core Features

• User authentication and role management
• Create, assign, and edit tasks
• View task lists by status or assignee
• Team dashboard for quick progress overview
• Clean UI with responsive layout

System Flow

User login → Create/assign task → Update status → Review task list → Dashboard summary

Setup

Clone repository
```bash
git clone https://github.com/krishfr/Task-Allocation---ISKCON-IT-Cell.git
cd Task-Allocation---ISKCON-IT-Cell
```

Install frontend dependencies
```bash
cd iskon-app
npm install
```

Access application
```bash
http://localhost:3000
```
